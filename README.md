### - Client Side

- [Sass](https://sass-lang.com/)

- [React Router](https://reactrouter.com/en/main)
- [React Bootstrap (optional)](https://react-bootstrap.netlify.app/)

- [MongoDB](https://www.mongodb.com/atlas/database)
- [Render](https://render.com/)
